# Project Quiz Master

installed 
- Node.js 
# Setup
unzip 
cd api
npm install   
cd ..
cd app
npm install
## Running
Start the API server:

In the api directory, run:

npm start

Running Frontend

In the app directory, run:

npm run dev
The UI  will start running at http://localhost:5173 
Admin account:
admin@gmail.com
## Features

- USER

- ADMIN
